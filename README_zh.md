# Carpet SRU Addition

[English](README.md)

Carpet SRU Addition 是一个适用于 Minecraft 1.21.11 的 [Carpet](https://github.com/gnembon/fabric-carpet) 扩展模组。

它包含一组面向 SRU 服务器的实用规则，包括悦灵行为微调、龙息可再生、TNT 随机动量控制、旁观者可见性调整以及配方开关。

- [English rules](docs/rules.md)
